Privacy Policy

1. Overview

BRUT is a web-based document utility developed and maintained under Bhavesh Rohilla Projects. This Privacy Policy describes how user information is collected, used, and protected when accessing or using BRUT.

By using BRUT, you agree to the practices described in this Privacy Policy.

2. Information We Collect

We collect only the information that is necessary to provide core functionality of the service.

2.1 Account and Login Information

The following data may be collected during account creation or login:

  • Email address
  • Authentication identifiers required for secure login

This information is used exclusively for:

  • User authentication
  • Account management
  • Security and access control

No additional personal data is collected beyond what is required for login and service operation.

3. File Processing and Storage

BRUT is designed with a privacy-first architecture.

  • Files processed using BRUT are not uploaded to or stored on BRUT servers.
  • All file operations are performed locally within the user’s browser or client-side environment.

3.1 Cloud Storage Integrations

Files are uploaded only when the user explicitly connects and authorizes one of the following third-party services:

  • Google Drive
  • Microsoft OneDrive

In such cases:

  • File transfers occur directly between the user and the connected cloud service
  • BRUT does not permanently store user files on its own servers
  • File access is limited strictly to the scope authorized by the user

4. Use of Third-Party Services

BRUT integrates with third-party platforms solely to enable optional features such as cloud file import and export.

These services operate under their own privacy policies and terms of service. Bhavesh Rohilla Projects does not control and is not responsible for the data handling practices of these third-party providers.

5. Data Security

Reasonable administrative and technical safeguards are implemented to protect user account information against unauthorized access, alteration, or disclosure.

While no method of transmission or storage is completely secure, BRUT takes appropriate measures to minimize risk.

6. Data Usage Limitations

Bhavesh Rohilla Projects does not:

  • Sell or rent user data
  • Analyze, monitor, or index user files
  • Store processed documents on its servers
  • Use user data for advertising or profiling purposes

User data is used solely to operate and improve the service.
